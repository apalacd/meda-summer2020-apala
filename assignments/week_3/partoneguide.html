HTML

HTML is used to descibe the content of web pages. HTML stands for HyperText Markup Language, and it is the primary language for websites, along with CSS, and optionally, JavaScript.     A webpage is split into two categories: Content and Design. HTML is responsible for the Content, so we will not be working with colors, font sizes or positioning, instead HTML is only used to describe what is displayed on the web page.

HTML SYNTAX

Just like we put together letters into groups to create words, HTML is grouped up into “tags”, a tag has a keyword that is surrounded by angle brackets ( < and > ). There are certain keywords that make up the language of HTML.
